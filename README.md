## INTRODUCTION TO DATA STRUCTURE 101

Welcome to my self-learning repository

This repository focuses on preparation for my DATA STRUCTURE course

## Overview

- This repository contains **30 practice problems** designed as a guideline for learning Data Structures progressively
- Each problem is implemented in two versions:
  - A guided version (skeleton code)
  - A completed version (solution)

## Folder Structure

The workspace contains two folders by default, where:

- `skeleton-code`: guided templates with hints and TODOs
- `solution`: complete implementations and reference solutions

## Base File

- `basis.cpp`: a core reference file located at the root directory, used as a fundamental guideline for this course
